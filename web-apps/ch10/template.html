§ 10.1 · Key Concept 10.1

Partial effects vs total effects

The bivariate coefficient for bedrooms is $23,667 — but add size to the model and it drops to $1,553. The "total" effect included size's contribution; the "partial" effect isolates what bedrooms adds beyond size.

Try this
  1. Bedrooms controlling for size: the coefficient drops from $23,667 to $1,553 — a 93% reduction. Size was doing most of the work.
  2. Switch focus to bathrooms, controlling for size. Does the same pattern hold?
  3. Try age controlling for size. The bivariate effect was already near zero — controlling barely changes it.
  4. Now control for bedrooms instead of size. The drop is smaller because bedrooms is a weaker confounder.
§ 10.2 – § 10.3 · Key Concepts 10.2 & 10.3

Correlation matrix heatmap

A correlation matrix shows all pairwise linear associations at a glance. High correlations between predictors (e.g., size-bedrooms r = 0.52) signal potential multicollinearity — the reason partial effects differ from total effects.

Try this
  1. Find the strongest predictor of price: size (r = 0.79). This is why the size-only model has the highest R².
  2. Find size-bedrooms (r = 0.52). This correlation is why bedrooms' coefficient shrinks in a multiple regression.
  3. Look at monthsold-bathrooms (r = −0.39). These two are negatively correlated in this sample.
  4. Which predictor pairs have near-zero correlation? These would remain relatively stable across bivariate and multiple regression.

§ 10.5 · Key Concept 10.5

Frisch-Waugh-Lovell theorem

The FWL theorem reveals how multiple regression isolates each variable's partial effect: residualize both Y and X on the controls, then regress the residuals. The slope equals the multiple regression coefficient exactly.

Try this
  1. Default: size controlling for bedrooms. The FWL slope (72.41) matches the multiple regression coefficient exactly.
  2. Switch target to bedrooms, control to size. The FWL slope ($1,553) matches price ~ bedrooms + size.
  3. Try bathrooms controlling for size. The FWL slope shows bathrooms' partial contribution after removing size's influence.
  4. The x-axis shows "Residualized X" — the parts of X that the control variable couldn't explain.
§ 10.6 – § 10.7 · Key Concepts 10.6 & 10.7

Model comparison — when more is less

R² can only go up as you add variables. Adjusted R², AIC, and BIC penalize complexity — they can go down when extra variables add noise rather than signal. Watch what happens as you build from 1 to 6 predictors.

Try this
  1. R² (cyan) rises from 0.618 to 0.651 — a modest 3.3 percentage-point gain for 5 extra variables.
  2. Adjusted R² (purple) falls from 0.603 to 0.555. The penalty for complexity exceeds the gain in fit.
  3. AIC rises from 668 to 676 — higher is worse. BIC rises even more (671 to 685) because it penalizes small-sample complexity more heavily.
  4. All four metrics agree: the 1-variable model (size only) is the best specification for n = 29.
§ 10.8 · Key Concept 10.8

Variance inflation factors — detecting multicollinearity

VIF measures how much a predictor's variance is inflated by correlation with the other predictors. VIF = 1 means no collinearity; VIF > 5 is moderate concern; VIF > 10 is severe. With n = 29 and 6 correlated predictors, the VIFs are alarming.

Try this
  1. Bedrooms has VIF = 57.8 and size has VIF = 40.1. Both far exceed the VIF > 10 threshold — severe multicollinearity.
  2. Even lotsize (VIF = 12.0) and monthsold (VIF = 12.8) exceed the threshold. With only 29 houses, 6 predictors is too many.
  3. This explains why only size is significant in the full model: multicollinearity inflates standard errors, making it harder to detect real effects.
  4. The remedy: drop redundant predictors (the parsimony principle from the model comparison widget).
§ 10.4

Regression diagnostics — actual vs predicted & residuals

A good model produces predictions close to actual values (points near the 45° line) and residuals that scatter randomly around zero (no patterns).

Try this
  1. Size only: points scatter around the 45° line with some spread. The residual plot shows no obvious pattern.
  2. Switch to the full model. The scatter tightens slightly (R² rises from 0.618 to 0.651) but the improvement is marginal.
  3. In the residual plot, look for fan shapes or curves. Random scatter means the model's assumptions are reasonable.
  4. Notice the house around $340,000 — it's consistently under-predicted across models.
