Key Concept 4.1: se(X̄) = s/√n. Quadrupling the sample size halves the SE. The curve above follows a 1/√n shape — precision improves rapidly at small n but yields diminishing returns at larger n.

t-Distribution Explorer

Key Concept 4.2: The t-distribution has fatter tails than the normal, reflecting extra uncertainty from estimating σ. At df = 4, the 5% critical value is 2.78 (much larger than 1.96). By df = 100, it is nearly indistinguishable from N(0, 1).
Try This
  1. Set df = 4. How much fatter are the tails compared to the normal? Note the t-critical value: 2.78 vs 1.96.
  2. Set df = 30. The t-critical is now ~2.04 — close to 1.96. The rule of thumb “n ≥ 30 is large enough” makes sense visually.
  3. Set df = 100. Can you even tell the difference from the normal? Look at the tail area beyond ±1.96 — it is nearly 5%.

Key Concept 4.4: t = (X̄ − μ0) / SE. The p-value is the probability of observing a result at least as extreme, assuming H0 is true. At μ0 = $40,000 the book gets t = 0.72 and p = 0.470 — not enough evidence to reject.
Try This
  1. Set μ0 = $40,000 (the book's example). The t-stat is ~0.72 and p ≈ 0.470. The shaded rejection regions are far from the t-stat marker. We fail to reject.
  2. Drag μ0 down to $35,000. Now the t-stat moves further from zero. At what μ0 does p first drop below 0.05? That boundary is the 95% CI endpoint.
  3. Set μ0 = $50,000. Now t is strongly negative. Rejection is clear. The p-value is tiny.

Try This
  1. Set μ0 = $40,000, Ha: μ > $40,000. The one-sided p-value is 0.235 — half of the two-sided 0.470. Still not significant.
  2. Now set μ0 = $38,000. The one-sided test rejects, but the two-sided does not. This illustrates the power advantage of directional tests.
  3. Toggle to Ha: μ < μ0. What happens when the t-stat is positive but you are testing in the wrong direction?

Key Concept 4.5: At μ0 = $3.81, t = −5.26 and p < 0.0001. The difference ($0.14/gallon) is statistically significant because the SE is tiny ($0.027). But is $0.14 practically significant? That depends on context — a commuter filling 15 gallons saves only $2.10.
Try This
  1. At H0 = $3.81 (the book's CA average), the t-stat is −5.26. Move H0 closer to the sample mean ($3.67). At what value does the test stop rejecting?
  2. Set H0 = $3.67 (the sample mean itself). The t-stat is ~0 and p ≈ 1.0. The rejection region is empty. Of course we cannot reject that μ equals X̄.
  3. Notice how the histogram clusters tightly around $3.50–$3.90. The small spread (SD ≈ $0.15) means even small departures from H0 are detectable.
§4.7

Inference for Proportions

Key Concept 4.9: se(p̂) = √(p̂(1−p̂)/n). The book's voter-intent example: 480/921 = 52.1% Democrats, SE = 0.0165, 95% CI = [48.9%, 55.3%]. Since 50% is inside the CI, we cannot reject H0: p = 0.50.
Try This
  1. Default: p̂ = 0.52, n = 921. The 95% CI includes 0.50, so the test fails to reject. This matches the book's voter example (z = 1.29, p = 0.199).
  2. Keep p̂ = 0.52 but increase n to 2,500. Now the CI tightens and 0.50 falls outside — the same proportion becomes significant with a larger sample.
  3. Set p̂ = 0.50 exactly. The z-stat is 0 and p = 1. The CI is perfectly centered on the hypothesized value.
