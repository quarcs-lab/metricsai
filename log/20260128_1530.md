## Problem Description

### Issue
In Results sections, redundant labels appeared before figure and table environments:

```latex
\textbf{Figure 2.2: Box Plot}    ← Redundant label (removed)
\begin{figure}[h]
\centering
\includegraphics[width=0.8\textwidth]{code_python/images/ch02_boxplot.png}
\caption{Box Plot of Earnings Distribution}  ← LaTeX provides numbering
\label{fig:ch02:boxplot}
\end{figure}
```

### Solution
Remove the standalone `\textbf{...}` lines since the LaTeX `\caption{}` command automatically provides:
- Proper numbering (Figure 2.2, Table 10.1, etc.)
- Consistent formatting
- Cross-reference support
- Inclusion in List of Figures/Tables
