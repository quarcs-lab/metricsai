## Work completed

### 1. Chapter 3 dashboard (`web-apps/ch03/dashboard.html`, 61.4 KB)

Built a single self-contained HTML dashboard covering Chapter 3 (The Sample Mean) sections 3.2–3.7 with 5 interactive widgets:

| Widget | Key Concept(s) | Interaction |
|---|---|---|
| Sampling Distribution & SE | 3.2, 3.3, 3.4 | Toggle: Coin Toss/Census Ages; Normal overlay on/off; SE bands off/±1/±2 |
| Sample Size Effect | 3.3, 3.4 | Slider n=5–500; Resimulate button; live Bernoulli simulation (500 samples) |
| Central Limit Theorem | 3.5, 3.6 | Dropdown: 5 population shapes; Slider n=2–100; side-by-side pop vs means |
| Estimator Properties | 3.7 | Toggle: Mean/Median/Trimmed 10%; Sliders: n, outlier contamination 0–20% |
| Weighted Means | 3.8 | Sliders: group means, population/sample fractions; bar chart comparison |

Tech stack: Same as Ch01/Ch02 (Plotly.js, brand palette, Inter + JetBrains Mono, light/dark theme, URL hash state, Reset buttons, Try-this blocks).

Simulation engine: Seeded PRNG (mulberry32) for reproducible sampling. Box-Muller for normal samples, Marsaglia-Tsang for gamma samples (census-like population).

Numerical verification — all values match the chapter:
- Coin toss: mean of 400 means = 0.4994, SD = 0.0863, theoretical SE = 0.0913
- Census ages: mean of 100 means = 23.78, SD = 3.76, theoretical SE = 3.72

## Decisions made

1. **All web-app files in chapter directory.** Build scripts and templates belong alongside the generated dashboard, not in a shared `scripts/` folder. Keeps each chapter self-contained and easier to navigate.
2. **Consistent naming:** `build.py` and `template.html` (not `build_chNN_webapp.py` and `chNN_webapp_template.html`). The directory already identifies the chapter.
3. **Simulation-based widgets for Ch03.** Unlike Ch01/Ch02 which primarily visualize pre-computed data, Ch03 uses live JavaScript simulation (seeded PRNG) for the sample-size, CLT, and estimator widgets. This lets students manipulate n and see results instantly.
4. **Five population shapes for CLT widget:** Bernoulli, Uniform, Exponential (right-skewed), Bimodal, and Census-like (Gamma distribution matching the book's census parameters).
5. **Outlier contamination slider for estimator widget.** Shows when the median outperforms the mean (high contamination), which is a key pedagogical question students ask.
