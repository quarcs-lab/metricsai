## Formatting Standards Applied

### Table Environment Structure
```latex
\begin{table}[h]
\centering
\caption{Table X.Y: Descriptive Title Explaining What Data Shows}
\label{tab:chXX:short-kebab-description}
\begin{tabular}{...}
\toprule
... header ...
\midrule
... data ...
\bottomrule
\end{tabular}
\end{table}
```

### Figure Environment Structure
```latex
\begin{figure}[h]
\centering
\includegraphics[width=0.8\textwidth]{path/to/image.png}
\caption{Figure X.Y: Descriptive Title Explaining What Visualization Shows}
\label{fig:chXX:short-kebab-description}
\end{figure}
```

### Key Changes
1. **Converted `\begin{center}` to proper environments:** All tables and figures now use proper LaTeX environments
2. **Added captions:** All items have descriptive `\caption{}` commands following the format "Table/Figure X.Y: Description"
3. **Added labels:** All items have `\label{}` commands for cross-referencing (format: `tab:chXX:name` or `fig:chXX:name`)
4. **Updated table rules:** Replaced `\hline` with booktabs commands (`\toprule`, `\midrule`, `\bottomrule`)
5. **Maintained decorative images:** Chapter visual summary images (e.g., `ch01_visual_summary.jpg`) left as decorative without figure environments
