# 2026-04-26 22:00 — Migrate Python Regressions from statsmodels to pyfixest

## Work Done

Migrated **all Python regression code** from `statsmodels` (`smf.ols()`) and `linearmodels` (`PanelOLS`) to **pyfixest** (`pf.feols()`), the Python port of R's fixest package.

### Why

- **Cross-language consistency**: Python `pf.feols()` mirrors R `fixest::feols()` — same formula syntax, same vcov options
- **Simpler API**: No `.fit()` step, no separate `cov_type`/`cov_kwds` — vcov is a single parameter
- **Built-in model comparison**: `pf.etable()` replaces manual table construction
- **Panel data without extra package**: `pf.feols('y ~ x | entity')` replaces linearmodels
- **Modern package**: Actively maintained, designed for econometrics

### Syntax Mapping (key changes)

| Before (statsmodels) | After (pyfixest) |
|---|---|
| `ols('y ~ x', data=df).fit()` | `pf.feols('y ~ x', data=df)` |
| `.fit(cov_type='HC1')` | `vcov='HC1'` |
| `.fit(cov_type='cluster', ...)` | `vcov={'CRV1': 'var'}` |
| `.fit(cov_type='HAC', ...)` | `vcov='NW', vcov_kwargs={...}` |
| `PanelOLS(y, X, entity_effects=True)` | `pf.feols('y ~ x \| entity')` |
| Manual 2SLS | `pf.feols('y ~ exog \| endog ~ inst')` |
| `model.params['x']` | `fit.coef()['x']` |
| `model.rsquared` | `fit._r2` |
| `model.fittedvalues` | `fit.predict()` |
