### 2. Maddison Project 2023 data

- `scripts/fetch_maddison_us.py` — downloads from Our World in Data's CSV mirror, extracts USA 1820→2022.
- `data/maddison_us_gdppc.csv` — 203 rows, $2,674 (1820) to $58,488 (2022) in 2011 PPP dollars.
- Used in the log-widget's time-series section to show the hockey-stick pattern (exponential growth → straight line after log).
- OLS slope on ln(GDP) = 1.56%/year; CAGR = 1.54%/year.
- Citation: Bolt & van Zanden (2024), *Journal of Economic Surveys*.

### 4. `web-app` skill (`.claude/skills/web-app/`)

A reusable agentic skill for creating chapter dashboards. Invoked via `/web-app chNN`.

Structure:
- `SKILL.md` — 5-phase workflow (understand → design → scaffold → implement → verify)
- `templates/base.html` — 691-line HTML scaffold with all shared CSS, JS helpers, theme, hash/reset layer

Tested: scaffold on ch99 (dry run, cleaned up); verify_app on ch02 (all checks passed).

## Decisions made

1. **Plotly.js** for charts (rich interactivity, single CDN, no build step).
2. **Embedded JSON** rather than live data fetches (works offline, no CORS).
3. **Single HTML file per chapter** — opens with double-click, no server needed.
4. **"Try this" prompts per widget** — highest-leverage pedagogical feature.
5. **3-way Level/Log/Growth toggle** for time-series widgets (eliminates confusing two-toggle interaction).
6. **Maddison 2023 long-run data** for the hockey-stick illustration (book's 1959-2020 series is too short).
7. **Widget snippets as Markdown** in the skill — forces adaptation, prevents blind copy-paste.
8. **Badge next to Colab badge** (same row, below thumbnail) rather than overlaid on the image.
