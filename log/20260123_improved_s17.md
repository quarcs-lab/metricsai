**Section 17.2: Panel Data**
- Panel data structure: multiple observations per individual over time

- Fixed effects eliminate time-invariant unobserved heterogeneity

**Section 17.5: Causal Inference: An Overview**
- Potential outcomes framework: Y_1i (treated), Y_0i (untreated)
- Individual treatment effect: Y_1i - Y_0i (unobservable)
- Average treatment effect (ATE): E[Y_1i - Y_0i]
- Average treatment effect on treated (ATET): E[Y_1i - Y_0i | D_i = 1]
- Randomized control trials (RCTs):
  - Random assignment ensures E[Y_0i | D_i=1] = E[Y_0i | D_i=0]
  - Simple difference estimator is unbiased for ATE
  - Gold standard for causal inference
- Difference-in-differences (DiD):
  - Natural experiments with treatment and control groups
  - Controls for time trends: (»≤_post - »≤_pre)_treatment - (»≤_post - »≤_pre)_control
  - Parallel trends assumption
  - Example: minimum wage effects on employment
- Regression adjustment:
  - Add control variables X to RCT or DiD
  - Improves precision by reducing residual variance
  - Can address imbalances in randomization
- Regression discontinuity design (RDD):
  - Treatment assigned based on threshold rule
  - Compare outcomes just above and below cutoff
  - Local causal estimate near threshold
  - Uses flexible polynomials or nonparametric methods
  - Example: test score cutoffs for program eligibility
- Local average treatment effects (LATE):
  - IV estimates causal effect for compliers
  - Compliers: induced into treatment by instrument
