# 2026-04-26 18:00 — Add Python | Stata Code Tabs to All 17 Web App Dashboards

## Work Done

Added a **Stata code tab** alongside the existing Python code in all 17 web app dashboards (ch01-ch17). Students can now toggle between Python and Stata to learn econometrics in their preferred language.

### Structural changes (all 17 dashboards)

- **Section header**: "Python Libraries and Code" → "Code Summary"
- **Nav anchor**: "Python code" → "Code"
- **Motivation paragraph**: Updated to be language-neutral
- **Tab bar**: Added `.code-tabs` pill toggle with Python (default) and Stata buttons
- **Two panels**: `panel-python` (existing code) and `panel-stata` (new Stata translation)
- **Element IDs**: `code-cheatsheet` → `code-python` (Python), `code-stata` (Stata)
- **Copy buttons**: Each panel has its own copy button targeting its code element
- **Colab link**: Python panel only; Stata panel shows "Paste into your Stata do-file editor"
- **CSS**: ~25 lines of tab styling (inline-flex pill toggle, active state with `--accent` color)
- **JS**: Tab-switching handler using `.code-tabs` class (deliberately NOT `.toggle-group` to avoid URL hash interference)
- **Prism.js**: Custom inline `Prism.languages.stata` grammar (~25 lines) covering 3 comment styles, compound strings, ~100 keywords, functions, numbers, operators

### Quality review — 7 bugs fixed

After writing all Stata code, a careful review found and fixed 7 bugs:

1. **CH05**: `%fmt` display directives in graph `title()` — invalid Stata syntax
2. **CH09**: `%6.4f` in graph title (log-linear plot) — same issue
3. **CH09**: `%4.2f 100*growth` in graph title (S&P 500) — same issue
4. **CH10**: `estimates table stats(aic bic)` — `aic`/`bic` not stored as `e()` scalars by `regress`
5. **CH15**: `vce(robust) beta` — potentially incompatible options; split into two regressions
6. **CH15**: `summarize earnings naive_pred corrected_pred` then `r(mean)` — reported wrong variable's mean
7. **CH17**: Pooled OLS / Fixed Effects labels swapped after `xtreg` and `estimates restore`

## Design Decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Tab class | `.code-tabs` (not `.toggle-group`) | Avoids triggering URL hash state listeners |
| Stata highlighting | Custom inline `Prism.languages.stata` | No official Prism Stata component exists |
| Colab link | Python tab only | Colab is Python-only |
| Section header | "Code Summary" | Language-neutral when tabs present |
| `beta` option | Separate from `vce(robust)` | Compatibility with older Stata versions |
