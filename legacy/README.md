# Legacy Archive

This directory contains archived files from the metricsAI project development process.
These files are preserved for historical reference but are no longer actively used.

## Contents

| Directory    | Description                                                                                  |
| ------------ | -------------------------------------------------------------------------------------------- |
| `backups/`   | Notebook backups from the standardization process (Feb 2026)                                 |
| `code_python/` | Original Python code files (pre-notebook versions)                                         |
| `code_r/`    | Original R code files                                                                        |
| `code_stata/` | Original Stata do-files                                                                     |
| `docs/`      | Historical documentation (template checklists, feature specs)                                |
| `scripts/`   | One-off scripts used during chapter standardization, case study additions, and early PDF experiments |

## Note

These files are kept for reference only. The active project uses:

- `scripts/` (root level) for reusable tooling
- `notebooks_colab/` for all chapter content
- `CLAUDE.md` for AI assistant instructions
